Spectrum V Project

Overview

In this project I recreated my own version of Artist: Ellsworth Kelly's 1969 spetrum V. using css and html.

Below I've left a walkthrough of how so that you could create your own, if you so wish to, by the end of the walkthrough you'll have created something like this using html and css.

Walkthrough

To re-create this project, you'll need to understand how to build a table using Html, and how to link an Html file to your CSS form. You'll also want to have a basic understanding of color theory, and how the CSS box model works.

CSS

Great! Now that we've completed our Html file, we can start working on styling it using CSS. To style our CSS, we'll want to learn how to select specific elements within our Html. Thankfully this isn't hard at all; in order to select an element, write it out and follow it by open and closed curly brackets. For example, if you are going to select a heading three then write h3{} within our curly brackets we can specify which elements will want to change.

Now that we have centered our page, we can start working on our table's color palette. We can change each 'td' cell individually by calling the assigned id, which we have assigned to it. Once we have selected our id, we can change its background color by either calling a hex code, an RGB code or simply writing the color we would like to change our background too.
